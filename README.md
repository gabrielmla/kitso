# savetv
necessário:
node (8.9.4)
bower (1.8.2)
mongodb
Express >=4.5

instalar dependencias:
npm install
bower install>
renomeie o arquivo db.js.backup para apenas db.js e substitua <login> e <senha> pelos valores respectivos ou opte por rodar um db local.

Para rodar a aplicação:
node server/app.js
